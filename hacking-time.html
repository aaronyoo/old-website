We are given an NES ROM which we can open in FCEUX (which is apparently the best NES emulator and comes with a built in debugger). We are given a story that the person that owns this device needs it back. They tell us that our first step is to connect to the grid so that we can “hack the device back”. They tell us that we need to create a segmentation fault. We can do this easily by entering AAAAAAAAAAAAAAAA.... as the login string. Our strategy here is as follows. Enter a string and search for it in the Hex Editor (that comes with FCEUX). From there we can set a watchpoint and see where our input is being read and checked. Here we go. Opening the hex editor after entering AAAAA:

We can see the string I input right there so let's click on the first byte of that string (the first 41) and set a “read breakpoint”. This opens up the debugger for us. Now let's go back to the ROM screen and try to submit our password (by pressing A). This causes the debugger to stop at the spot where it tripped our breakpoint.

I figured that we need to understand the assembly so I looked up the manual of 6502 Assembly https://en.wikibooks.org/wiki/6502_Assembly. There are a bunch of different addressing modes so it turned out to be easier to just decode the instructions as I proceeded instead of doing them on the fly. After about an hour of decoding I think I have the logic of down. At a high level, the assembly is performing some manipulation on the inputs (using memory locations near 0x955E for the computation and storing the results at 0x001E. The program then procedurally checks each output value (after all of them have been generated) and if one of them is not equal to 0, it returns with a failure immediately. Also we know that the password has to be 0x18 = 24 characters long. Here is some Python3 pseudocode that I wrote as I was going along and reading the assembly: 

from bitstring import *
stack = []

a = BitArray('0x41')
a.rol(3)
stack.append(a)
a = MEM[3B]
a.ror(2)
MEM[3B] = a
a = stack.pop()
a += MEM[3B]
a ^= MEM[955E + Y]
MEM[3B] = a
a.rol(4)
a ^= MEM[9576 + Y]
MEM[001E + Y] = a
Y += 1
if (Y < 24):    
        jump to top

Y = 0
A = MEM[001E + Y]
if (A != 0):
        A = 0   
        return ## fail
Y += 1
if (Y != 24):    
        # jump to line 25 (check if the next Y is zero)
A = 1
return # success

The b variable has replaced MEM[3B] and the lists represent the 24 interesting hex values starting at that address. This is so that we can perform the offset operation with and index and we can simulate the program memory. I simply copied these values out of the hex editor.

But now comes the question of how do we brute force a solution. Technically we could try every ascii character in order to brute force a solution. However, let's take this opportunity to learn about Microsofts Z3 Theorem Prover. Apparently this type of theorem prover can help us in this situation. Here is the summary page that I read to help me use Z3: https://www.cs.tau.ac.il/~msagiv/courses/asv/z3py/guide-examples.htm

Well.. after many days of trying, turns out that Z3 is just too hard to use and too above my comprehension level right now. So we are going to try brute forcing the solution instead. I was stuck for about 1 day because one of my memory values was incorrect (in my list) which caused my program to find the wrong password. Fixing this immediately gave me the flag: 

from bitstring import *
from string import *
import sys
import copy

MEM_955E = [0x70, 0x30, 0x53, 0xA1, 0xD3, 0x70, 0x3F, 0x64, 
            0xB3, 0x16, 0xE4, 0x04, 0x5F, 0x3A, 0xEE, 0x42,
            0xB1, 0xA1, 0x37, 0x15, 0x6E, 0x88, 0x2A, 0xAB]
MEM_9576 = [0x20, 0xAC, 0x7A, 0x25, 0xD7, 0x9C, 0xC2, 0x1D, 
            0x58, 0xD0, 0x13, 0x25, 0x96, 0x6A, 0xDC, 0x7E,
            0x2E, 0xB4, 0xB4, 0x10, 0xCB, 0x1D, 0xC2, 0x66]

def checkPassword(s):
    b = BitArray('0x00')
    for y in range(len(s)):
        a = BitArray(format(ord(s[y]) & 0xff, '#04x'))
        a.rol(3)
        b.ror(2)
        a = BitArray(format(((a.int + b.int) & 0xff), '#04x'))
        a ^= BitArray(format(MEM_955E[y], '#04x'))
        b = copy.copy(a)
        a.rol(4)
        a ^= BitArray(format(MEM_9576[y], '#04x'))
        if (a.int != 0):
            return False
    return True

key = ""
while len(key) < 24:
    for c in printable:
        if checkPassword(key + c):
            key += c
            print(key)
            break
